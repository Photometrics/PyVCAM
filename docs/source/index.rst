Welcome to PyVCAM's documentation!
==================================

This library integrates the `PyVCAM`_ Python3.X wrapper for the `Teledyne Photometrics`_ `Prime BSI Scientific CMOS (sCMOS) camera`_ 
with `ARTIQ`_ real-time control software as a `Network Device Support Package (NDSP)`_.

.. _PyVCAM: https://github.com/Photometrics/PyVCAM
.. _Teledyne Photometrics: https://www.photometrics.com
.. _Prime BSI Scientific CMOS (sCMOS) camera: https://www.photometrics.com/products/prime-family/primebsi
.. _ARTIQ: http://m-labs.hk/experiment-control/artiq
.. _Network Device Support Package (NDSP): https://m-labs.hk/artiq/manual/developing_a_ndsp.html
