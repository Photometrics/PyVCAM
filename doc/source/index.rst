Welcome to PyVCAM's documentation!
==================================

This library integrates the `PyVCAM <https://github.com/Photometrics/PyVCAM>`_ Python3.X wrapper for the `Teledyne Photometrics <https://www.photometrics.com/>`_
`Prime BSI Scientific CMOS (sCMOS) camera <https://www.photometrics.com/products/prime-family/primebsi>`_ 
into the `ARTIQ <http://m-labs.hk/experiment-control/artiq/>`_ (`Github <https://github.com/m-labs/artiq>`_)
environment via the usage of `Network Device Support Packages (NDSPs) <https://m-labs.hk/artiq/manual/developing_a_ndsp.html>`_.
